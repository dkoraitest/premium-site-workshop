# Чек-лист перед запуском

Финальная проверка перед тем как дать ссылку клиенту/публике.

## Контент

- [ ] Все тексты вычитаны, нет lorem ipsum
- [ ] Все ссылки работают (нет 404)
- [ ] Телефоны и email кликабельны (tel:, mailto:)
- [ ] Изображения загружаются, нет broken images

## Технические

- [ ] Сайт открывается на mobile (iPhone SE, Android)
- [ ] Сайт открывается на desktop (Chrome, Safari, Firefox)
- [ ] Нет горизонтального скролла на мобильных
- [ ] Favicon отображается в табе браузера
- [ ] Title и description уникальны для каждой страницы

## SEO

- [ ] JSON-LD schema добавлена и валидна
- [ ] og:image существует и корректного размера (1200x630)
- [ ] sitemap.xml доступен
- [ ] robots.txt доступен и разрешает индексацию
- [ ] canonical URL корректен
- [ ] llms.txt добавлен

## Аналитика

- [ ] Яндекс.Метрика установлена и счётчик подтверждён
- [ ] GA4 установлен (если нужен)
- [ ] Цели настроены (CTA click, form submit)
- [ ] Cookie consent работает (если нужен)

## Формы

- [ ] Форма отправляется и уведомление приходит в Telegram
- [ ] Валидация работает (попробуй отправить пустую форму)
- [ ] Сообщение об успехе отображается
- [ ] Сообщение об ошибке отображается
- [ ] Кнопка блокируется при отправке

## Performance

- [ ] Hero-видео загружается за разумное время (< 5 сек на 4G)
- [ ] Lazy loading для контента ниже fold
- [ ] Изображения в WebP (или сжатые JPEG)
- [ ] Нет неиспользуемых скриптов/стилей

## Безопасность

- [ ] Пройден чек-лист security.md
- [ ] `.env` в `.gitignore`
- [ ] HTTPS работает
- [ ] Privacy policy доступна
