# Product Scroll — Horizontal Pin + Parallax + Reveal

## Для кого
Продуктовые лендинги (электроника, авто, мебель), e-commerce, недвижимость, архитектурные бюро.

## Что получается
- Горизонтальная pinned-секция (вертикальный скролл → горизонтальное движение)
- Parallax-слои (элементы двигаются с разной скоростью)
- Clip-path reveal для изображений (круг/прямоугольник раскрывается при скролле)
- Smooth scroll (Lenis)
- Продуктовый фокус: крупные фото, минимум текста, кинематографичность

## Стек
- GSAP ScrollTrigger (pin + horizontal scroll)
- Lenis (smooth scroll)
- CSS clip-path (reveal анимации)
- Опционально: Spline для 3D-модели продукта

## Промпты

### 1. Создание сайта
> Создай лендинг для [ПРОДУКТ/БРЕНД]. Стиль: Apple-like продуктовая презентация. Hero: полноэкранное фото продукта с parallax-эффектом (фон двигается медленнее текста). Далее — горизонтальная pinned-секция: при вертикальном скролле секция прибивается к экрану, 4-5 карточек (фото + текст) прокручиваются горизонтально. Используй GSAP ScrollTrigger pin:true. Секция галереи: изображения раскрываются через clip-path: circle() при скролле. На мобильных — обычный вертикальный layout.

### 2. Parallax усиление
> Добавь multi-layer parallax: фоновый слой двигается со скоростью 0.3x, средний 0.6x, передний 1x от скорости скролла. Реализуй через GSAP ScrollTrigger с разными значениями y для каждого слоя. Для hero-секции: текст уходит вверх быстрее чем изображение.

### 3. 3D-модель (опционально)
> Встрой Spline 3D-модель продукта: [SPLINE_URL]. Iframe в контейнере с aspect-ratio, position absolute, z-index 0. Текст поверх. На мобильных — заменить на статичное изображение.
