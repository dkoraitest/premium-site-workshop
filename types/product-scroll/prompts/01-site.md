# Шаг 1: Создание сайта

## Что делаем
Создаём продуктовый лендинг с горизонтальным скроллом и reveal-анимациями.

## Промпт для Claude Code

> Создай лендинг для [ПРОДУКТ/БРЕНД]. Стиль: Apple-like продуктовая презентация. Hero: полноэкранное фото продукта с parallax-эффектом (фон двигается медленнее текста). Далее — горизонтальная pinned-секция: при вертикальном скролле секция прибивается к экрану, 4-5 карточек (фото + текст) прокручиваются горизонтально. Используй GSAP ScrollTrigger pin:true. Секция галереи: изображения раскрываются через clip-path: circle() при скролле. На мобильных — обычный вертикальный layout. Smooth scroll через Lenis.

## Что заменить
- `[ПРОДУКТ/БРЕНД]` — название продукта или бренда

## Ожидаемый результат
Продуктовый лендинг с горизонтальным скроллом, parallax и раскрывающимися изображениями.
