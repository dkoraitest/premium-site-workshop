# Checkpoint 2: Все секции

**Когда:** ~35 мин после начала build-фазы (01:00 от начала воркшопа)

## Критерии готовности

- [ ] 4-5 секций на месте (Hero, About, Features, Cases/Portfolio, Contact)
- [ ] Навигация работает (якорные ссылки скроллят к секции)
- [ ] На мобильных (375px): одна колонка, всё читаемо
- [ ] Нет горизонтального скролла на мобильных
- [ ] Footer с контактной информацией
- [ ] Контент заменён на свой (не lorem ipsum)

## Типичные проблемы

| Проблема | Решение |
|----------|---------|
| Горизонтальный скролл | `overflow-x: hidden` на `body`, проверь `width` и `padding` элементов |
| Секции слипаются | Добавь `padding: 80px 0` на каждую `section` |
| Навигация не скроллит | Проверь `id` на секциях и `href="#section-id"` на ссылках |
| Grid ломается на мобильных | `grid-template-columns: 1fr` в `@media (max-width: 768px)` |
| Карточки разной высоты | `align-items: stretch` на grid-контейнере |
| Текст выходит за границы | `word-wrap: break-word` и `max-width: 100%` на контейнерах |

## Быстрый фикс
> Проверь responsive: открой в 375px ширину. Исправь все проблемы: горизонтальный скролл, обрезанный текст, неработающую навигацию. Карточки — в одну колонку на мобильных.
