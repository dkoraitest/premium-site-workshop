# Checkpoint 1: Hero-секция

**Когда:** ~15 мин после начала build-фазы (00:40 от начала воркшопа)

## Критерии готовности

- [ ] Hero занимает полный экран (100vh или 100dvh)
- [ ] Заголовок + подзаголовок читаемы на любом фоне
- [ ] CTA-кнопка видна и кликабельна
- [ ] На мобильных (375px): текст не обрезается, кнопка доступна
- [ ] Нет ошибок в консоли браузера

## Типичные проблемы

| Проблема | Решение |
|----------|---------|
| Текст не читается на фоне | Добавь overlay: `background: rgba(0,0,0,0.4)` между фоном и текстом |
| Hero меньше экрана | `height: 100vh` или `min-height: 100dvh` (dvh лучше для мобильных) |
| Шрифт не загрузился | Проверь `<link>` на Google Fonts в `<head>`, проверь имя шрифта в CSS |
| CTA не видна на мобильных | Уменьши `font-size` заголовка, проверь `padding` и `position` |
| Видео не воспроизводится | Атрибуты: `muted autoplay playsinline loop`. Файл < 3MB |
| Gradient скучный | Добавь 2-3 цвета, используй `radial-gradient` вместо `linear-gradient` |

## Быстрый фикс
Если hero совсем не работает, попроси Claude Code:
