**Формат:** 8-этапный pipeline — все этапы, ключевые глубоко, остальные обзорно

## Production Pipeline

### 00:00–00:10 — Этап 1: Бизнес-анализ

Обзорно. Кто ЦА, какую боль решаем, выбор типа сайта.
- Skills: /brainstorming → /grill-me
- Материал: pipeline/01-business-analysis.md
- Результат: BUSINESS_ANALYSIS.md

### 00:10–00:15 — Этап 2: ТЗ на сайт

Обзорно. Структура, секции, контент-план, стек.
- Skill: /grill-me для прожарки ТЗ
- Материал: pipeline/02-site-spec.md
- Результат: SITE_SPEC.md

### 00:15–00:25 — Этап 3: Инфраструктура

Глубоко. Clone репо, установка skills, выбор типа, создание папки.
- Материал: pipeline/03-infrastructure.md
- Результат: готовая среда с skills

### 00:25–01:25 — Этап 4: Реализация (60 мин)

Глубоко. Основной этап — строим сайт по промптам.
- Skills: taste-skill + gsap-skills + orchestrating-gsap-lenis
- Промпты: types/{тип}/prompts/01-site.md, 02-effects.md
- CP1 (00:40): Hero готов
- CP2 (01:00): Все секции на месте
- CP3 (01:25): Эффекты работают

### 01:25–01:30 — Этап 5: Контент

Обзорно. Замена placeholder'ов на реальный контент.
- Материал: pipeline/05-content.md

### 01:30–01:35 — Этап 6: Проверка (QA)

Обзорно. Формы, responsive, кроссбраузерность.
- Материал: pipeline/06-qa-check.md
- Запускать в отдельной сессии Claude Code

### 01:35–01:45 — Этап 7: Security + SEO + Legal

Глубоко. SEO-промпт + security чек-лист + privacy policy.
- Промпт SEO: types/{тип}/prompts/03-seo.md
- Security: checklists/security.md (ОТДЕЛЬНАЯ сессия!)
- Материал: pipeline/07-security-seo.md

### 01:45–01:55 — Этап 8: Деплой + Аналитика

Глубоко. Деплой на Cloudflare Pages, формы → Telegram, Метрика + GA4.
- Промпты: types/{тип}/prompts/04-forms.md, 05-deploy.md
- Аналитика: common/analytics.md
- Материал: pipeline/08-deploy-analytics.md

### 01:55–02:00 — Финал

2-3 волонтёра показывают результат. Before-launch чек-лист. Q&A.

## Заметки ведущего

- **Этапы 1-2 и 5-6** — обзорные, показываем подход. Участники углубятся дома.
- **Этапы 3-4 и 7-8** — основное время, строим руками.
- **Кто-то отстаёт** — дай промпт, не объясняй. «Скопируй вот это».
- **Claude зависает** — 30 сек, потом новая сессия.
- **Эффекты не работают** — консоль. 90% — не подключён GSAP/ScrollTrigger/Lenis.
- **Время гибкое** — быстро? Углубляемся в контент. Медленно? Пропускаем QA.
