# Воркшоп: Премиум-сайт за один спринт

**Дата:** 14 мая 2026
**Длительность:** 2 часа
**Формат:** Обзор → выбор типа → все строят параллельно (единый flow)
**Репо:** github.com/dkoraitest/premium-site-workshop

---

## Агенда

### 00:00–00:15 — Intro + Showcase (15 мин)

**Цель:** Участники видят конечный результат и выбирают свой тип.

1. Открой `resources/showcase.md` — покажи 1-2 примера из каждого типа
2. Объясни 4 типа (1 мин каждый):
   - **Video Hero** — кинематографичный scroll-scrub (luxury, tech, недвижимость)
   - **SaaS Aurora** — gradient + glass + bento (SaaS, AI, fintech)
   - **Agency Kinetic** — text reveal + cursor (агентства, портфолио)
   - **Product Scroll** — horizontal pin + parallax (продукты, e-commerce)
3. Стек: vanilla HTML/CSS/JS + GSAP + Lenis. Без фреймворков — всё просто.
4. "Сейчас каждый выберет свой тип и за час мы соберём рабочий сайт"

### 00:15–00:25 — Setup (10 мин)

**Цель:** Все готовы к работе.

Инструкция на экране:
```
git clone https://github.com/dkoraitest/premium-site-workshop.git
cd premium-site-workshop

# Установи taste-skill (опционально но рекомендуется)
npx skills add https://github.com/Leonxlnx/taste-skill

# Создай папку для своего сайта
mkdir my-site && cd my-site

# Открой Claude Code
```

Каждый выбирает тип → открывает `types/{type}/prompts/` → готов.

### 00:25–00:40 — Шаг 01: Создание сайта (15 мин)

**Все:** открывают `types/{тип}/prompts/01-site.md`, копируют промпт, заменяют placeholder.

**Ведущий:** делает то же самое параллельно — свой тип.

Подсказка: "Замените [БИЗНЕС/БРЕНД] на свой. Остальное Claude сделает."

### 00:40 — Checkpoint 1: Hero ✓

Быстрая проверка: "Поднимите руку у кого hero на весь экран. У кого проблемы — зовите."
См. `checkpoints/cp1-hero.md`

### 00:40–01:00 — Продолжение шага 01 (20 мин)

Допиливаем секции, контент, responsive. Кто быстрый — уже переходит к шагу 02.

### 01:00 — Checkpoint 2: Все секции ✓

"Откройте на мобильном (DevTools → 375px). Нет горизонтального скролла? Секции на месте? Отлично."
См. `checkpoints/cp2-sections.md`

### 01:00–01:25 — Шаг 02: Premium-эффекты (25 мин)

**Все:** открывают `types/{тип}/prompts/02-effects.md`, копируют промпт.

Это ключевой момент — эффекты создают wow. Дайте Claude время.

### 01:25 — Checkpoint 3: Эффекты ✓

"Прокрутите страницу. Анимации работают? Курсор кастомный? Видео крутится?"
См. `checkpoints/cp3-effects.md`

### 01:25–01:35 — Шаг 03: SEO (10 мин)

**Все:** `types/{тип}/prompts/03-seo.md`

Быстрый шаг — Claude добавит мета-теги, JSON-LD, sitemap, robots.txt.

### 01:35–01:45 — Шаг 04: Форма → Telegram (10 мин)

**Все:** `types/{тип}/prompts/04-forms.md`

**Важно:** кто заранее создал бота — подключает форму. Остальные пропускают, можно после.

### 01:45–01:55 — Шаг 05: Деплой (10 мин)

**Все:** `types/{тип}/prompts/05-deploy.md`

Рекомендация: Cloudflare Pages. За 5 минут сайт в интернете.

### 01:55–02:00 — Финал (5 мин)

1. "Кто хочет показать результат?" — 2-3 волонтёра шарят экран
2. Чек-лист для домашней работы: `checklists/before-launch.md`
3. Ссылка на репо — забирайте, пользуйтесь
4. Вопросы

---

## Заметки ведущего

- **Если кто-то отстаёт** — помоги с конкретным промптом, не с объяснением. "Скопируй вот это, вставь в Claude Code".
- **Если Claude зависает** — подожди 30 сек, если не отвечает — новая сессия с тем же промптом.
- **Если эффекты не работают** — проверь консоль. 90% проблем — не подключён GSAP/ScrollTrigger/Lenis.
- **Время гибкое** — если build идёт быстро, добавь SEO+формы. Если медленно — пропусти формы.
