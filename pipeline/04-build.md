# Этап 4: Реализация

Основной этап — строим сайт. Самый длинный блок на воркшопе (60 мин).

**Мульти-агентный подход:** после каждого checkpoint запускаем агентов-ревьюеров для проверки качества. Агенты лежат в `.claude/agents/` — они уже в репо.

---

## Агенты проекта

В `.claude/agents/` лежат 3 агента. Они запускаются из основной сессии Claude Code:

| Агент | Что делает | Когда запускать |
|-------|-----------|----------------|
| `design-reviewer` | Скриншот + оценка визуала (типографика, spacing, цвета, premium) | После CP1, CP2, CP3 |
| `code-reviewer` | Качество кода (GSAP, performance, accessibility, чистота) | После CP2, CP3 |
| `responsive-checker` | Проверка 375px, 768px, 1440px | После CP2, CP3 |

Чтобы запустить агента, скажи Claude Code:
> Запусти агента design-reviewer для проверки текущего состояния сайта.

---

## Порядок работы

### Шаг 01: Hero + базовая архитектура

Откройте `types/{ваш-тип}/prompts/01-site.md`, скопируйте промпт, замените placeholder.

**Checkpoint 1 (~15 мин):** Hero на весь экран, CTA видна, mobile ok.
→ Если проблемы: `checkpoints/cp1-hero.md`
→ **Запусти:** `design-reviewer` — проверит визуал hero

### Шаг 01 (продолжение): Все секции

Допиливаем секции, контент, responsive.

**Checkpoint 2 (~35 мин):** 4-5 секций, навигация, нет горизонтального скролла.
→ Если проблемы: `checkpoints/cp2-sections.md`
→ **Запусти:** `design-reviewer` + `responsive-checker` + `code-reviewer`
→ Исправь найденные FAIL перед переходом к эффектам

### Шаг 02: Premium-эффекты

Откройте `types/{ваш-тип}/prompts/02-effects.md`, скопируйте промпт.

**Checkpoint 3 (~60 мин):** Основной эффект работает, анимации плавные, mobile fallback.
→ Если проблемы: `checkpoints/cp3-effects.md`
→ **Запусти:** все 3 агента — финальная проверка перед переходом к контенту

---

## Цикл разработки

```
Написал код → Checkpoint → Запустил агентов → Исправил FAIL → Двигаемся дальше
```

Этот цикл повторяется на каждом checkpoint. Агенты находят проблемы, которые глаз может пропустить: мелкий шрифт на мобильном, забытый `alt` у изображения, дёрганую анимацию.

---

## Типичные проблемы на этом этапе

| Проблема | Решение |
|----------|---------|
| GSAP не подключён | CDN: `cdn.jsdelivr.net/npm/gsap@3/dist/gsap.min.js` |
| ScrollTrigger не работает | Отдельный файл + `gsap.registerPlugin(ScrollTrigger)` |
| Lenis конфликтует | `lenis.on('scroll', ScrollTrigger.update)` |
| Анимация дёргается | Анимируй только `transform` и `opacity` |
| Мобильные тормозят | `matchMedia('(max-width: 768px)')` — отключи тяжёлое |
